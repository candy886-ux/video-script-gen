# 视频脚本一键生成

> 播放量总低迷？直接帮你拉高完播率

## 前置依赖

```bash
pip install pandas openpyxl
```

用 `web_search` 搜索该主题的爆款视频案例，分析其脚本结构和亮点；用 `web_fetch` 抓取参考视频页面信息。

### 能力2：检索该领域热门视频风格（web_search）

用 `web_search` 搜索相关信息。

### 能力3：撰写完整脚本（开场hook+内容+结尾CTA）

用 `write_to_file` 生成文件。

### 能力5：适配目标平台时长和风格

## 使用流程

### 步骤 1：收集用户需求

向用户确认以下信息（如果未主动提供）：
- 视频主题和类型（教程/vlog/带货/科普/口播）
- 目标平台（抖音/B站/YouTube/小红书）
- 视频时长（15秒/1分钟/3分钟/10分钟）
- 面向的受众群体
- 参考视频链接（如有）

### 步骤 2：检索外部信息

执行以下搜索获取真实数据：

```
web_search("[主题] 爆款视频脚本 [平台]")
web_search("[主题] video script template")
web_search("[平台] 热门视频 [主题] 分析")
```

确保获取到以下资源：
- 爆款视频参考链接
- 选题分析

### 步骤 3：运行脚本处理数据

```bash
python3 scripts/video_script_gen_tool.py run \
  --input "用户提供的输入" \
  --output "/path/to/output_file"
```

读取脚本输出的结果，确认数据处理成功。

### 步骤 4：生成最终产出

基于脚本输出和搜索到的资源，用 `write_to_file` 生成以下文件：

- **视频脚本文档**
- **分镜表**

输出格式要求：Markdown 脚本 + 分镜表

### 步骤 5：汇总交付

向用户展示：
1. 生成的文件路径和内容摘要
2. 搜集到的资源链接列表
3. 关键发现和建议

## 注意事项

- 所有数据必须来自 `web_search` / `web_fetch` 的真实搜索结果，**严禁编造数据**
- 数据缺失时标注"数据不可用"而非猜测
- 报告必须保存为文件（`write_to_file`），不能只在对话中输出
- 建议结合人工判断使用，AI 分析仅供参考
